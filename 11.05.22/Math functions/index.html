CSS Math Functions

The CSS math functions allow mathematical expressions to be used as property values. Here, we will explain the calc(), max() and min() functions.

The calc() Function

The calc() function performs a calculation to be used as the property value.

Syntax

calc(expression)

Description about expression of parantheses: Required. A mathematical expression. The result will be used as the value. The following operators can be used: + - * /

Example

Width with calc(100% - 300px)

The max() Function

The max() function uses the largest value, from a comma-separated list of values, as the property value.

Syntax

max(value1, value2, ...)

Description about value1, value2, ... of parantheses: Required. A list of comma-separated values - where the largest value is chosen

Example

Width with max(40%, 200px)

The min() Function

The min() function uses the smallest value, from a comma-separated list of values, as the property value.

Syntax

min(value1, value2, ...)

Description about value1, value2, ... of parantheses: Required. A list of comma-separated values - where the smallest value is chosen

Example

Width with min(80%, 800px)

The clamp() Function

The clamp() CSS function clamps a value between an upper and lower bound. clamp() enables selecting a middle value within a range of values between a defined minimum and maximum.

Syntax

clamp(value1, idealValue, value3)

Description about value1, idealValue, value3 of parantheses: Required. It takes three parameters: a minimum value, a preferred value, and a maximum allowed value. The clamp() function can be used anywhere a <length>, <frequency>, <angle>, <time>, <percentage>, <number>, or <integer> is allowed.
