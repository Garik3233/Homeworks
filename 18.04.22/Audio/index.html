Audio files

HTML <audio> tag, supports 3 types of audio files

  • mp3 - MIME-type audio/mpeg
  • wav - MIME-type audio/wav
  • ogg - MIME-type audio/ogg

<audio> tag attributes

src
Specifies the URL of audio file.

autoplay
Attached audio will play automatically when it loads.

control
Specifies what controls should display with the audio player (i.e. play, stop, pause button).

loop
Specifies audio file should be repeat continuously.

muted
Specifies audio file should be muted.

preload
Specifies how to audio file should load.

Used attributes: controls, autoplay, muted
